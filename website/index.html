Supercharge your skills

At Stackademy, you will learn how to plan, develop and deploy dynamic web applications using the Ruby on Rails framework.

With an experienced instructor and tons of practical exercises, Stackademy gets you up to speed with everything you need to successfully work on software projects:

  • Ruby and the Ruby on Rails framework
  • Databases
  • Command-line interface
  • Versioning
  • Testing
  • Deployment on cloud infrastructure
  • HTML & CSS
  • JavaScript

Learn in Europe's tech hot spot

Stackademy is 100% based and run in Europe's most dynamic startup hub Berlin, giving you the opportunity to meet with startups, tech companies and people in tech from all over the world.

Yes, we have set up the course to be serious about coding.
Still, you will find Berlin a great place to be at: Be it fashion, clubs and music, arts, third-wave coffee or startups, Berlin is buzzing with activity and has a rich history to show.

Berlin also hosts a range of technology user groups, from frontend design to JavaScript and Django, and we are partnering with some of the city's finest startups to add exciting guest lectures to the course.

In short: you're at the right place.
